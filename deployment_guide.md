
# Инструкция по развертыванию на Ubuntu Server

### 1. Подготовка системы
Обновите пакеты и установите Docker:
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install docker.io docker-compose -y
sudo systemctl enable --now docker
```

### 2. Запуск приложения
Перейдите в папку с проектом и выполните:
```bash
sudo docker-compose up -d --build
```

### 3. Доступ
Приложение будет доступно по IP-адресу вашего сервера на порту **80**.
Например: `http://192.168.1.100`

### Примечания для сканирования:
Поскольку это веб-интерфейс, реальное сканирование сети (nmap/arp-scan) требует бэкенда с привилегиями root. В текущей версии интерфейс имитирует работу сканера. Для подключения реального сканера необходимо развернуть FastAPI бэкенд из архитектурного описания.
